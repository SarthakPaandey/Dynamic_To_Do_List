# 📝 Dynamic To-Do List

A dynamic, interactive to-do list web application built with vanilla JavaScript. Deployed on GitHub Pages.

## Live Demo

[sarthakpaandey.github.io/Dynamic...](https://sarthakpaandey.github.io/Dynamic_To_Do_List/)

## Overview

This project is a lightweight to-do list app that allows users to add, complete, and remove tasks dynamically without any page reloads. Built purely with HTML, CSS, and JavaScript.

## Features

- Add new tasks dynamically
- Mark tasks as complete
- Remove tasks
- Clean, responsive UI
- No frameworks or dependencies

## Tech Stack

| Component | Technology |
|-----------|------------|
| Markup    | HTML       |
| Styling   | CSS        |
| Logic     | JavaScript |
| Hosting   | GitHub Pages |

## Getting Started

Open `index.html` in a browser, or visit the [live demo](https://sarthakpaandey.github.io/Dynamic_To_Do_List/).
